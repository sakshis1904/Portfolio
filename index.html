Prescripto

A comprehensive full-stack appointment scheduling system built with the MERN stack that streamlines healthcare bookings. The platform features secure JWT authentication, an intuitive admin dashboard for managing doctors and approvals, and integrated Razorpay payment processing for seamless transactions. It automates coordination workflows and ensures reliable, secure access for both patients and healthcare providers.

DietAssist

DietAssist is a full-stack AI web application that provides personalized diet and workout plans using voice interaction. The core idea was to make fitness guidance natural, fast, and accessible, especially for users who prefer speaking over typing. The system listens to user voice inputs like health goals, body details, dietary preferences, and activity level, processes them using AI, and instantly generates customized fitness plans.

UniQuery

A complaint management platform designed to enhance student living conditions by enabling convenient time-specific maintenance visit bookings. Built with the MERN stack and Tailwind CSS, it features a real-time admin dashboard for tracking and approving complaints, significantly improving issue resolution efficiency. The system optimizes complaint workflows and reduces unresolved issues through streamlined communication between students and administrators.

CareerNest

A full-featured job portal platform connecting job seekers with recruiters through personalized dashboards. Job seekers can create profiles, upload resumes, and apply to postings, while recruiters can post opportunities, shortlist candidates, and schedule interviews. Built with the MERN stack, it implements Clerk Authentication for secure role-based access and uses optimized MongoDB queries for efficient job matching and filtering.

ClimeSense

ClimeSense is a MERN-based weather monitoring app with secure JWT authentication. It fetches real-time weather and location data using an external Weather API. Users can log in, search locations, view temperature, humidity, and forecasts, making it a simple and reliable personal climate tracker.
